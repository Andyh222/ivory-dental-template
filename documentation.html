The template includes two colour themes. style.css is the clean blue/white version. style-colour.css is the warm forest green, coral and gold version. To switch themes, change the <link> tag in <head>.

Getting Started

Open dental-site.html in a browser to preview the template. For the best experience during development, serve the files via a local server rather than opening directly as a file:// URL:

VS Code — Live Server (recommended)

Install the Live Server extension, right-click dental-site.html and choose Open with Live Server.

Python (if installed)

cd your-project-folder
python3 -m http.server 8000
# Then open http://localhost:8000/dental-site.html

Customising Colours

All colours are CSS custom properties at the top of style.css. Change these tokens and the entire site updates automatically:

:root {
  --c-blue:    #007AFF;   /* Primary accent — buttons, links, eyebrows */
  --c-blue-dk: #005EC4;   /* Hover state for primary accent */
  --c-ink:     #1A1A1A;   /* Dark text and dark section backgrounds */
  --c-offwhite:#F5F5F7;   /* Light section backgrounds */
  --c-mint:    #E8F5F0;   /* Very light tint used on image wrappers */
  --c-muted:   #6E6E73;   /* Body/secondary text */
  --c-border:  #D2D2D7;   /* Dividers and card borders */
}
To use the alternative colour theme, change the stylesheet link in <head> from style.css to style-colour.css. The colour theme uses a forest green, coral and gold palette.

Updating Opening Hours

The live open/closed status widget reads from an hours object in assets/js/main.js. Find this block near the bottom of the file and edit the values (24-hour format). Set a day to null for closed:

const hours = {
  1: { open: 8,  close: 19 }, // Monday    8am–7pm
  2: { open: 8,  close: 19 }, // Tuesday
  3: { open: 8,  close: 19 }, // Wednesday
  4: { open: 8,  close: 19 }, // Thursday
  5: { open: 8,  close: 19 }, // Friday
  6: { open: 9,  close: 16 }, // Saturday  9am–4pm
  0: null,                     // Sunday — closed
};

The status uses London time (Europe/London) automatically, so it works correctly for users in any timezone.

Updating the Map

Find the <iframe> in the Location section and replace the src with an embed URL for your own address:

  1. Go to maps.google.com and search for your address
  2. Click Share → Embed a map
  3. Copy the src URL from the iframe code provided
  4. Paste it into the template's iframe src attribute

Updating Pricing Plans

The three pricing cards are in the <!-- PLANS --> section of the HTML. Each card has a .plan-price element — update the number and the small <span> for the pence/cents. Update the .plan-period text and the .plan-features list items as needed.
